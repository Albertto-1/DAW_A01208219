Preguntas:

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet World Wide Web
El internet es una red de redes inmensa en donde todas las computadoras que acceden al internet están conectadas WWW es la herramienta que nos ayuda a accesar a la información que está en la internet

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Método HTTP: Propósito
GET Se usa para solicitar la representación de un recurso.
HEAD Al igual que GET, también solicira un recurso pero no su representación.
POST Se usa para enviar información. Se usa al llenar formularios.
PUT Este método puede actualizar y crear contenido.
PATCH Se utiliza para actualizar parte del contenido de un sitio, dejando al resto intacto.
DELETE Elimina un resurso.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Yo pienso que se debe de usar el método POST porque es el que envía información.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Se utiliza el método GET porque estás soicitando la representación de a página.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

El código 200 significa que la página cargó de forma correcta

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

No porque el error 404 significa o que la página ya no existe o que se tecleó mal el url por lo que no se encuentra la página.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Este error sí lo tiene que corregir el desarrollador porque significa un error en el código que hizo que el seridor no pueda general a página.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Significa que el atributo aún es soportado pero que es mejor no usarlo ya. Un ejemplo es hgroup que se usaba para agrupar el título y el subtítulo de algo. Otros elementos depreciados son: center, command, scoped, etc.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 permite incluir nuevo elementos como SVG y MathML, también se introducen nuevos elementos como section, article, aside, header, footer... y nuevos atributos como media, charset, autofocus, placeholder...

Tiene mejoras API (getElementsByClassName() y innerHTML)

¿Qué componentes de estructura y estilo tiene una tabla?

tr Son las filas de una tabla.
th Títulos de una tabla.
td Contenido de una tabla.

También tienen elementos como cellpading, align, border, height, width, etc.

¿Cuáles son los principales controles de una forma HTML5?

Los principales controles de una forma HTML5 son number, range, date, color, search... Existen más pero esos son algunos.
