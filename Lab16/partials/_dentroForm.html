Preguntas:

¿Qué es ODBC y para qué es útil?

Open Database Connectivity

Hace posible el acceder a cualquier dato desde cualquier aplicación, sin importar qué sistema de gestión de bases de datos (DBMS) esté almacenando los datos

¿Qué es SQL Injection?

Es un ataque a la base de datos en donde un usuario obtiene acceso a la misma y modifica la estructura.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Tener solo un usuario que pueda modificar la estructura de la base de datos.
